Snap Eats is a full stack Food ordering application that supports:

Customers browsing restaurants, ordering food, tracking orders

Restaurant owners managing menus, categories, ingredients, events

Super Admin managing all customers and restaurants

Secure authentication with JWT

Role-based access: Customer, Restaurant Owner, Super Admin

Real-time restaurant dashboard, menu operations, ingredient stock mgmt

Redux state management, REST APIs

FrontEnd:

React.js

Redux Toolkit

Material UI (MUI)

React Router

Backend:

Spring Boot (Java)

Spring Security + JWT

Hibernate + JPA

MySQL Database


Software Requirements:

Visual Studio Code

IntelliJ Idea

Java 17 or above

Maven 3.8+

Spring Boot 3.x

MySQL Server 8+

React

Postman

Hardware Requirements:

CPU - i5 or above
RAM - 8GB + (fast Build)
Storage - 3+ GB


How to run the backend:
1. Update application.properties
  - create database 
  - username update
  - password
  - update stripe key

2. Install dependencies  if needed

3. Run mvn spring-boot:run


How to run the frontend:
1. Install dependencies 
2. npm start
